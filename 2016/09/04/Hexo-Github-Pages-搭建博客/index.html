Hexo + Github Pages 搭建博客

Posted on Sep 4 2016

什么是 Hexo ?

Hexo 是一个快速、简洁且高效的博客框架。

快速安装 Hexo

Hexo官方文档 可以帮助我们快速安装 Hexo 。

需要两个应用程序

  • Node.js
  • Git

Node.js 安装

  • 在 Node.js官网 直接下载安装包 (V4.5.0 LTS) 打开安装即可。

Git 安装

我们可以通过 Homebrew 来安装 Git 。
在 终端 输入以下代码即可安装 Git 。

  • /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" #安装 Homebrew
  • brew install git #安装 Git

安装 Hexo

完成上述步骤即可在终端输入npm install -g hexo-cli 安装 Hexo 。

本地化 Hexo

  1. 创建一个文件夹。例如我创建了一个名为 Blog 的文件夹。
  2. 打开终端，输入以下命令。
  • cd Blog # 有关 hexo 的命令都要先切换到 Blog文件夹，后面将不再提示。
  • hexo init
  • npm intall

完成以上命令，Hexo会在 Blog 文件夹中建立所需文件。

预览建立的网站。

我们可以预览我们建立的网站，继续在 终端 输入以下命令。

  • hexo g
  • hexo s

随后我们在 Safari 打开 http://localhost:4000/ 即可预览我们建立的网站。但是这个网站只能我们自己本地看，所以我们需要部署 Hexo 到 Github Pages 才行。
最后别忘了在 终端 按下 control + C 来退出。

部署 Hexo 到 Github Pages

首先我们需要一个 Github 账号，如果还没有的话，则需要到 Github 注册一个。
注册完成之后我们就可以开始创建自己的 Github Pages。

创建 Github Pages

  1. 点击右上角的 + 选择 New repository。

  2. 在 Repository name 填写 yourname.github.io 。其中 yourname必须与你建立的账户名一致。（与左边 Owner 名相同）
    Description 可以填写你喜欢的，这里我填写了Blog。点击 Creat respository 完成创建。

  3. 创建完成后选择 Setting ，下拉到 GitHub Pages 选择 Launch automatic page generator → Countinue to layouts → Publish page 。

  4. 至此， Github Pages 创建完成。你可以在 yourname.github.io 查看你刚刚建立的 Github Pages。

  • npm install hexo-deployer-git --save #安装扩展
  • hexo d #部署

打开你自己的博客网址 yourname.github.io 查看。

站点配置文件及主题配置文件的修改

Blog 文件夹里面有两个 _config.yml 配置文件。其中一个上边打开修改过，称为站点配置文件。还有一个在 themes/你的主题名字 里面，称之为主题配置文件。
站点配置文件的属性在Hexo官网文档可以看到。
主题配置文件的修改下面会讲到。
下边教大家如何修改主题。

挑选喜欢的主题

Hexo主题有很多，大家可以自己去挑一个自己喜欢的，这里给大家举个例子。
这里我选择了MeiliDu，点击图片进入主题的 Github。

下拉找到 git 命令


PS：这个界面有介绍如何修改主题配置文件的。

然后我们打开终端，输入

  • hexo clean
  • git clone https://github.com/HoverBaum/meilidu-hexo.git themes/meilidu

这样我们就可以把 MeiliDu 主题下载下来了。打开 Blog→themes 就可以看到 meilidu 文件夹。

启用新主题

  1. 打开站点配置文件，下拉找到 theme 把后面修改为 meilidu 。

  2. 打开终端，执行以下命令。

  • cd themes/meilidu
  • git pull
  • hexo g #记得切换到 Blog文件夹
  • hexo s

然后就可以在 http://localhost:4000/ 查看新主题了。
看完记得control + C退出，然后hexo d部署到 Github Pages 。

这里有个小问题

你需要自己创建 About 界面。

这里参考 创建“关于我”界面

打开 终端 输入

  • hexo new page "about"

可以看到 sourse 文件夹里已经多了一个 about 文件夹，里面有一个 index.md 文件，打开在下面输入你想在此界面显示的内容即可。

如何发布新文章

思路是先在本地创建一个新文章，然后再部署上去。

创建新文章

打开 终端 输入命令。

  • hexo new "the title of your blog"

即可创建新文章。

你可以在 source → posts 里面找到你刚刚创建的文章。

部署上 Github Pages

当你完成该文章后打开 终端，

  • hexo g
  • hexo d

即可完成部署。当然你也可以hexo s先在本地查看。

Tips

下面是关于 hexo 的有关提示。

常用命令简写

  • hexo g == hexo generate
  • hexo d == hexo deploy
  • hexo s == hexo server
  • hexo n == hexo new
